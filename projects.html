Description

Python CLI app that uploads files (asynchronously) to MirrorAce using their API.

The intent was to handle files of any filesize and upload them in parallel, with options to upload entire directories, either as a single archive or as individual files.

mirour

  • CLI (cligen)

Description

A rewritten version of mirror-up, in Nim

The motivation behind this project was to learn Nim and to take a look at code I had written before college and perceive my improvement.

As Nim is a compiled language, this allows me to have a single binary that can target various platforms, so if any development is to be done, it will be done in this package.

Repository

Description

A little Python CLI app that provides a League of Legends champion build by scraping the web.

The goal was to not have to open a browser tab (or be advertised to) to quickly check a build before a match. The HTML is requested and parsed sequentially, so it's somewhat slow (but still faster than opening a browser, probably).
